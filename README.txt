IVAN KUANG PORTFOLIO - QUICK START GUIDE

1. HOW TO RUN:
   - Install "Live Server" extension in VS Code.
   - Right-click "index.html".
   - Select "Open with Live Server".

2. HOW TO EDIT TEXT/PROJECTS:
   - Open "index.html".
   - Scroll down to the "CONFIGURATION & DATA" section (Line ~80).
   - Change the text in quotes (e.g., PROFILE_NAME, PROFILE_BIO).
   - Add new projects to the PROJECTS list following the example.

3. HOW TO ADD PDFs:
   - Create a folder named "pdfs" in the same folder as index.html.
   - Put your PDF files inside it (e.g., "my-report.pdf").
   - In index.html, change the 'link' property of a project to:
     link: './pdfs/my-report.pdf'

Enjoy your new portfolio!
